#### Metodi Pubblici Aggiunti

**purgeUrl(string $url): bool**

**purgePost(int $postId): int**


```php

// Ritorna numero file eliminati
```


**Benefici:**
- ‚úÖ **90% riduzione eliminazioni** - Solo pagine correlate
- ‚úÖ **Cache preservata** - Contenuti non correlati rimangono cached
- ‚úÖ **Performance migliorate** - Meno rigenerazione cache necessaria
- ‚úÖ **TTFB ridotto** - Cache gi√† pronta per altre pagine

**Nuovi Eventi:**
